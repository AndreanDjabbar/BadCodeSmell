The Dispensible

Terjadi ketika terdapat sebuah bagian dari code yang tidak diperlukan. Dan ini harus dikeluarkan dari code.

Secara umum, The Dispensible memiliki beberapa jenis seperti: The Comment, Duplicate Code, Lazy Class, Data Class, Dead Code, dan Speculative Generality.

The Comment

Terjadi ketika terdapat sebuah code yang diiringi dengan comment untuk penjelasan code secara berlebihan atau tidak jelas. Contoh: 


                    public class Example {
                        // Menghitung dan mengembalikan hasil penjumlahan dua bilangan
                        // @param a Bilangan pertama
                        // @param b Bilangan kedua
                        // @return Hasil penjumlahan a dan b
                        public int add(int a, int b) {
                            return a + b;
                        }
                    }
Duplicate Code

Terjadi ketika terdapat dua code yang terlihat hampir sama atau bahkan sama. Contoh: 


                    public class Calculation {
    
                        // Menghitung luas persegi panjang
                        public int calculateRectangleArea(int length, int width) {
                            return length * width;
                        }
                        
                        // Menghitung luas segitiga
                        public double calculateTriangleArea(int base, int height) {
                            return 0.5 * base * height;
                        }
                        
                        // Menghitung volume balok
                        public int calculateRectangleVolume(int length, int width, int height) {
                            return length * width * height;
                        }
                        
                        // Menghitung volume kerucut
                        public double calculateConeVolume(int radius, int height) {
                            return (Math.PI * radius * radius * height) / 3;
                        }
                        
                        // Method duplicate untuk menghitung luas segitiga
                        public double calculateTriangleAreaDuplicate(int base, int height) {
                            return 0.5 * base * height;
                        }
                    }
Lazy Class

Terjadi ketika terdapat sebuah class yang memiliki cost time terlalu tinggi ataupun terlalu mahal.

Data Class

Terjadi jika terdapat sebuah class yang hanya memiliki getter dan setter. Ini akan mengakibatkan sebuah class tidak dapat berfungsi secara independen atau mandiri. Contoh: 


                    public class Employee {
                        private String name;
                        private int age;
                        private double salary;
                    
                        // Konstruktor
                        public Employee(String name, int age, double salary) {
                            this.name = name;
                            this.age = age;
                            this.salary = salary;
                        }
                    
                        // Getter dan setter untuk semua atribut
                        public String getName() {
                            return name;
                        }
                    
                        public void setName(String name) {
                            this.name = name;
                        }
                    
                        public int getAge() {
                            return age;
                        }
                    
                        public void setAge(int age) {
                            this.age = age;
                        }
                    
                        public double getSalary() {
                            return salary;
                        }
                    
                        public void setSalary(double salary) {
                            this.salary = salary;
                        }
                    }
Dead Code

Terjadi ketika terdapat bagian dari code yang merupakan variabel, parameter, method, ataupun class yang sudah tidak digunakan lagi karena sudah usang.

Speculative Generality

Terjadi ketika terdapat sebuah class, method, ataupun parameter yang tidak digunakan.
