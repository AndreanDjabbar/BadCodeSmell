Object Oriented Abuser

Terjadi ketika terdapat pola design yang berlebihan dalam menggunakan konsep OOP tanpa alasan yang jelas atau dengan cara yang tidak tepat.

Secara umum, Object Oriented Abuser memiliki beberapa jenis seperti: Switch Statement, Temporary Field, Refused Bequest, dan Alternative Class with Different Interface.

Switch Statement

Terjadi ketika terdapat penggunaan switch operator ataupun if statement yang sangat rumit atau kompleks. Contoh: 


                    public enum FoodType {
                        PIZZA,
                        BURGER,
                        SUSHI
                    }
                    
                    public class FoodOrderProcessor {
                        
                        public void processOrder(FoodType foodType) {
                            switch (foodType) {
                                case PIZZA:
                                    preparePizza();
                                    break;
                                case BURGER:
                                    prepareBurger();
                                    break;
                                case SUSHI:
                                    prepareSushi();
                                    break;
                                default:
                                    System.out.println("Unknown food type");
                            }
                        }
                        
                        private void preparePizza() {
                            System.out.println("Preparing pizza...");
                        }
                        
                        private void prepareBurger() {
                            System.out.println("Preparing burger...");
                        }
                        
                        private void prepareSushi() {
                            System.out.println("Preparing sushi...");
                        }
                    }
Temporary Field

Terjadi ketika terdapat sebuah element yang hanya digunakan pada keadaan tertentu saja dan tidak akan berguna jika di luar keadaan tersebut.

Refused Bequest

Terjadi ketika terdapat Penggunaan inheritance antara superclass dan subclass yang tidak menyeluruh. Biasanya subclass akan menyembunyikan function inheritancenya. Contoh: 


                    // Superclass
                    public class Animal {
                        public void eat() {
                            System.out.println("Animal is eating...");
                        }

                        public void sleep() {
                            System.out.println("Animal is sleeping...");
                        }

                        public void makeSound() {
                            System.out.println("Animal is making a sound...");
                        }
                    }

                    // Subclass
                    public class Dog extends Animal {
                        public void bark() {
                            System.out.println("Dog is barking...");
                        }
                    }
Alternative Class with Different Interface

Terjadi jika terdapat dua 2 buah class yang memiliki fungsi identik atau serupa namun memiliki nama fungsi yang berbeda. Contoh: 


                    public class Square {
                        private double side;
                    
                        public Square(double side) {
                            this.side = side;
                        }
                    
                        public double calculateArea() {
                            return side * side;
                        }
                    }
                    
                    public class Rectangle {
                        private double width;
                        private double height;
                    
                        public Rectangle(double width, double height) {
                            this.width = width;
                            this.height = height;
                        }
                    
                        public double computeArea() {
                            return width * height;
                        }
                    }
