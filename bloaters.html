The Bloaters

Terjadi ketika terdapat sebuah pola struktur atau design yang menjadi besar atau berlebihan sehingga akan membuat software tersebut menjadi lambat atau tidak efektif.

Secara umum, The Bloaters memiliki beberapa jenis seperti: Long Method, Large Class, Primitive Obsession, dan Long Parameter.

Long Method

Terjadi ketika ada sebuah method yang memiliki baris code yang sangat banyak. Biasanya code tersebut bisa lebih dari 10 baris code.

Large Class

Terjadi ketika terdapat sebuah class yang memiliki atribut, method, atau code yang terlalu panjang.

Primitive Obsession

Terjadi pada saat kita menggunakan tipe data primitif secara berlebihan dengan mengesampingkan atau mengabaikan tipe data yang lebih kuat. Contoh: 


                    public class Order {
                        private String orderId;
                        private String customerName;
                        private String shippingAddress;
                        private double totalPrice;
                    
                        public Order(String orderId, String customerName, String shippingAddress, double totalPrice) {
                            this.orderId = orderId;
                            this.customerName = customerName;
                            this.shippingAddress = shippingAddress;
                            this.totalPrice = totalPrice;
                        }
                    
                        // Getters and setters
                    }
Long Parameter

Terjadi jika terdapat sebuah method yang memiliki parameter yang terlalu banyak. Biasanya memiliki lebih dari 3 parameter. Contoh:
