Change Preventer

Terjadi ketika terdapat pola design pada code yang dirancang dengan begitu kompleks sehingga untuk membuat perubahan pada sebuah code atau program, akan menjadi sulit dan mahal. Contohnya ada bagian code yang rumit dan mempunyai hubungan yang berkaitan erat dengan bagian code lain.

Secara umum, Change Preventer memiliki beberapa jenis seperti: Divergent Change, Shotgun Surgery, dan Parallel Inheritance Hierarchies.

Divergent Change

Terjadi ketika terdapat bagian dari code yang memiliki tanggung jawab yang sangat luas atau memiliki ketergantungan yang kuat pada banyak bagian lain dari sistem.

Shotgun Surgery

Terjadi ketika pada saat kita ingin memodifikasi beberapa bagian, namun akan berpengaruh juga ke bagian lain. Contoh: 


                    public class ShotgunSurgery {
                        // Class 1
                        static class Customer {
                            private double discountRate;
                    
                            public Customer(double discountRate) {
                                this.discountRate = discountRate;
                            }
                    
                            public double calculatePrice(double price) {
                                return price - (price * discountRate);
                            }
                        }
                    
                        // Class 2
                        static class Product {
                            private double discountRate;
                    
                            public Product(double discountRate) {
                                this.discountRate = discountRate;
                            }
                    
                            public double calculateDiscount(double price) {
                                return price * discountRate;
                            }
                        }
                    
                        // Class 3
                        static class Order {
                            private double discountRate;
                    
                            public Order(double discountRate) {
                                this.discountRate = discountRate;
                            }
                    
                            public double applyDiscount(double totalPrice) {
                                return totalPrice - (totalPrice * discountRate);
                            }
                        }
                        public static void main(String[] args) {
                            double discountRate = 0.1;
                            Customer customer = new Customer(discountRate);
                            Product product = new Product(discountRate);
                            Order order = new Order(discountRate);
                            // Some operations with the classes
                        }
                    }
Parallel Inheritance Hierarchies

Terjadi pada saat kita ingin membuat sebuah subclass untuk suatu class, namun kita harus membuat subclass untuk class lainnya juga. Contoh: 


                    public class ParallelInHerit {
                        // Employee Type Hierarch
                        class EmployeeType {
                            public String getTitle() {
                                return "Employee";
                            }
                        }
                    
                        class ManagerType extends EmployeeType {
                            @Override
                            public String getTitle() {
                                return "Manager";
                            }
                        }
                    
                        class DirectorType extends EmployeeType {
                            @Override
                            public String getTitle() {
                                return "Director";
                            }
                        }
                    
                        // Employee Benefit Hierarchy
                        class EmployeeBenefit {
                            public String getBenefits() {
                                return "Basic Benefits";
                            }
                        }
                    
                        class ManagerBenefit extends EmployeeBenefit {
                            @Override
                            public String getBenefits() {
                                return "Basic Benefits + Bonus";
                            }
                        }
                    
                        class DirectorBenefit extends EmployeeBenefit {
                            @Override
                            public String getBenefits() {
                                return "Basic Benefits + Bonus + Stock Options";
                            }
                        }
                    
                    }
